# Syst-me-de-Tra-abilit-Alimentaire-Blockchain-IoT-du-Champ-au-Consommateur
Ce système révolutionnaire combine la Blockchain et l'IoT pour garantir une transparence totale, du champ jusqu'à l'assiette du consommateur. Des capteurs intelligents enregistrent en temps réel les données critiques (température, origine, humidité), créant ainsi un passeport numérique infalsifiable pour chaque produit.
